hacer paginas web pasandole un word

Estado: Completado

Características principales

Conversión automática de documentos Word (.docx) a HTML

Extracción de imágenes del documento original y organización en carpetas

Formato inteligente que detecta títulos, párrafos y bloques de código

Edición avanzada del contenido extraído antes de generar el HTML final

Plantillas personalizables para manuales técnicos y proyectos

Gestión de metadatos (descripción, fecha, tecnologías, estado del proyecto)

Actualización automática de archivos JSON para catálogos de proyectos/manuales

Previsualización del resultado antes de guardar

Modo edición para modificar contenido previamente generado

Tecnologías utilizadas

Python como lenguaje principal

Tkinter para la interfaz gráfica de usuario

python-docx para procesar archivos de Microsoft Word

Beautiful Soup para analizar y manipular HTML

JSON para almacenar metadatos de proyectos y manuales

Cómo funciona

Selecciona un documento Word con el contenido de tu manual o proyecto

Extrae el contenido para editarlo en la interfaz

Formatea elementos como títulos, párrafos o bloques de código según necesites

Configura metadatos como nombre, descripción o tecnologías utilizadas

Selecciona una imagen de portada opcional para tu proyecto o manual

Genera el HTML que se guardará en la estructura de carpetas de tu portafolio

Actualiza automáticamente los archivos JSON que catalogan todos tus proyectos

También hay un manual para que lo veas en funcionamiento. Puedes ir haciendo clic aquí

Características avanzadas

Estados de proyecto: Marca tus proyectos como "En desarrollo", "Completado", "Planificación", etc.

Tecnologías utilizadas: Etiqueta tus proyectos con las tecnologías implementadas

Bloques de código con formato: Detecta automáticamente URLs, comandos y fragmentos de código

Botones interactivos: Incluye botones para copiar código o editarlo en línea

Metadatos SEO: Genera automáticamente etiquetas meta para mejorar el SEO

Integración con portafolio web

La herramienta está diseñada para integrarse perfectamente con tu portafolio web, generando:

Archivos HTML basados en plantillas personalizables

Estructura de carpetas organizada para proyectos y manuales

Archivos JSON actualizados que sirven como "base de datos" para el listado de proyectos

Esta herramienta me ha permitido mantener mi portafolio técnico actualizado sin tener que editar manualmente el código HTML, automatizando la creación de contenido a partir de documentos Word y manteniendo un formato consistente en todos los proyectos y manuales.

El código de la herramienta estará publicado, pero con algunos cambios. Voy a quitar la parte de manuales técnicos y proyectos, y la dejaré con lo básico. Haré que cree automáticamente las carpetas necesarias para colocar los estilos, la plantilla HTML que se debe usar y algunas funciones adicionales. Una vez esté listo, podrás probarlo aquí.
